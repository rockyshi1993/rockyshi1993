### 🚀 Core projects
- **🔧 [vext](https://github.com/vextjs/vext)** — Modern full‑stack Node.js framework with convention‑based routing, plugin lifecycle, service injection, schema validation, and OpenAPI generation.  
- **📐 [schema-dsl](https://github.com/vextjs/schema-dsl)** — JavaScript DSL for data validation and runtime checks with i18n and DB schema export capabilities.  
- **🔁 [monSQLize](https://github.com/vextjs/monsqlize)** — Query adapter that maps SQL‑style queries to MongoDB operations to simplify migrations and multi‑database compatibility.  
- **⚡ [ndsk](https://github.com/vextjs/ndsk)** — nodeStack full‑stack framework focused on reducing boilerplate and accelerating development.  
- **🧩 Utilities** — cacheHub, rate limiting, permission core, and other small libraries that solve common backend needs.
